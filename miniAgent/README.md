miniAgent is a web-based Agent that conveniently enables the workflow from requirements to target image generation.

It comes with a built-in Pyodide sandbox tool, implementing an automated workflow of  **User <-> Large Model <-> Tool** .

**It solves the problem that large models can only return plotting code but cannot generate the plot in one step.**

## 关于miniAgent

**快速配置测试：**

```
1. 获取个人Deepseek 或 OpenAI密钥
2. 在miniAgent右上角单击模型配置
3. 左侧栏上部点击添加Skill文件，将项目的SKILL.md文件配置进去
4. 配置后在Skill方框内，将Skill旗下的reference全部导入
5. 左侧栏下部导入数据，即可在右侧Agent对话框进行测试
```
