Step 02

02Visual Idiom

i
Name — Sunburst Diagram（旭日图 / 径向树图），又称 radial space-filling hierarchy。
ii
Marks — 弧段（arc / annular sector）。每一个弧段代表层级树中的一个节点；环心代表树根。
iii
Visual variables — ① 角度编码数值大小（最精确通道）；② 径向距离编码层级深度；③ 色相编码顶层分类（洲）；④ 明度编码同分类内的子项区分；⑤ 位置（顺时针起始角）继承自父节点，保持上下文。
iv
Interaction — Click-to-zoom（点击下钻 / 中心回溯）支持层级聚焦；Hover 触发 tooltip 与高亮，承担细节按需查询；Legend filter 支持类别筛选；Linked detail panel 实现联动视图（linked highlighting / brushing 思想）。
v
Data structure — 严格的 tree（层级 / hierarchy）结构 — 每节点恰有一父节点，叶节点携带定量数值，非叶节点的值 = 子节点之和。
vi
Tasks — ① 比较整体组成（part-to-whole）；② 在层级间逐层下钻；③ 识别贡献度最大的子类；④ 跨同级分支做相对量级比较。
vii
Scalability — 较好支持 3–5 层、每层 ≤ 20 个子节点的中等规模层级；过深或过细时外环弧度极小，文字与角度判读失效。对极度稀疏 / 极度不平衡的树效果差。
Step 03

03Perceptual Principles

i
Preattentive features — 颜色（hue）、明度（luminance）与封闭形状（closure）在 200ms 内即可被并行处理。本设计中 hover 高亮 + 同源弧段保留高亮、其余降低不透明度形成强 pop-out 效果，引导注意力到当前路径；中心数字使用 serif 高对比字形成视觉锚点。
ii
Colour — 采用 7 个色相的离散类别色 + 同色相多明度梯度，避开红绿同时承担类别意义，整体在 deuteranopia / protanopia 下仍可凭明度区分；深色背景 (#0b0d12) 与暖色弧段保持 WCAG AA 级以上对比；语义上"碳排放"采用暖色系（琥珀-赭红-土黄）以承载主题情绪。
iii
Gestalt — ① 邻近性：同父节点的子弧紧邻排列；② 相似性：同色相暗示同分类；③ 共同区域：环状结构本身构成"封闭层"；④ 连续性：径向从内向外形成自然阅读流；⑤ 图形-背景：深背景 + 噪点纹理压低环境，让弧段稳定浮于前景。
